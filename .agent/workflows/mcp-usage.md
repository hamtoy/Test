# MCP 도구 활용 지침

## 1. Perplexity (검색/연구)

**언제 사용:**

- API 사용법 또는 SDK 문서 확인 필요 시
- Validation 에러, 타입 에러 발생 시
- 라이브러리 버전 호환성 문제 발생 시
- 최신 best practice 확인 필요 시

**규칙:** 스킵하거나 우회하기 전에 반드시 검색 먼저!

## 2. Sequential Thinking (복잡한 분석)

**언제 사용:**

- 아키텍처 설계 결정
- 버그 근본 원인 분석
- 여러 솔루션 비교/평가
- 리팩토링 전략 수립

## 3. SonarQube

**언제 사용:**

- 코드 품질 이슈 조회
- 커버리지 확인
- 중복 코드, 보안 취약점 확인

## 4. GitHub MCP

**언제 사용:**

- PR 생성/조회/리뷰
- 이슈 관리
- 코드 검색 (전체 GitHub)
- 브랜치/커밋 관리

## 5. Redis MCP

**언제 사용:**

- 캐시 데이터 조회/설정
- 키 패턴 검색

## 6. Notion MCP

**언제 사용:**

- Notion 페이지/데이터베이스 접근
- 문서 검색 및 업데이트
